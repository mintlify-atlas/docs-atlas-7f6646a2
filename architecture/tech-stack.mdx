The Hack Western Platform is built on the T3 Stack, a modern, type-safe web development stack optimized for full-stack TypeScript applications.

## Core framework

### Next.js 15.5.9

The foundation of the application, providing server-side rendering, routing, and API routes.

- **React 19.2.3**: Latest React with concurrent features
- **React Compiler**: Experimental compiler for automatic optimization
- **Pages Router**: Traditional routing pattern with `pages/` directory
- **Turbopack**: Next-generation bundler for development

## Database and ORM

### Drizzle ORM 0.41.0

Type-safe SQL ORM with zero-cost abstractions.

- **drizzle-orm**: Core ORM functionality
- **drizzle-kit**: CLI for migrations and introspection
- **drizzle-zod**: Schema validation with Zod
- **@auth/drizzle-adapter**: NextAuth adapter for Drizzle
