<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your development environment running in minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the Hack Western repository and navigate into the directory.
      
      ```bash
      git clone https://github.com/hackwestern/hackwestern.git
      cd hackwestern
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required packages using npm.
      
      ```bash
      npm install
      ```
      
      <Note>Node.js version 21 or higher is required. We recommend using [nvm](https://github.com/nvm-sh/nvm) to manage Node versions.</Note>
    </Step>
    
    <Step title="Set up environment variables">
      Get a copy of `.env` from a web lead or create your own. Generate a `NEXTAUTH_SECRET` with:
      
      ```bash
      openssl rand -base64 32
      ```
      
      Add this and other required variables to your `.env` file.
    </Step>
    
    <Step title="Start the database">
      Launch your local PostgreSQL database using Docker.
      
      ```bash
      npm run db:start
      npm run db:migrate
      ```
      
      <Accordion title="Optional: Seed the database">
        Populate your database with fake data for testing:
        
        ```bash
        npm run db:seed
        ```
      </Accordion>
    </Step>
    
    <Step title="Launch the development server">
      Start the Next.js development server with Turbopack.
      
      ```bash
      npm run dev
      ```
      
      Your app will be running at `http://localhost:3000`
    </Step>
  </Steps>
</div>
